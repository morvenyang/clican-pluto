使用帮助


安装流程:

  • 1, 通过Apple Store搜索《天王星》并下载安装该软件，目前可以支持所有iPad,iPad mini,iPhone4/4s/5,iTouch4/5
  • 2, 安装后运行应用,该应用会自动去服务器端注册并更新脚本
  • 3, 在设置中修改服务器地址,http://clican.org:9001是测试服务器 http://clican.org是正式服务器
  • 4, 打开Apple TV，访问Clican->本地服务器，就能够浏览iOS上运行的本地内容了。或者在TT的个人链接里设置http://clican.org:9001/appletv/local.xml以直接访问

功能列表:
  • 1, 支持通过iOS设备来代理下载视频内容，以解决播放优库和迅雷等相关视频内容卡顿、缓冲的问题
  • 2, 可以通过iOS设备来登录迅雷，以解决迅雷登录问题。目前登录后的迅雷帐号只能在Clican站点下使用
  • 3, 可以通过iOS设备来浏览相关视频内容，收藏视频，在iOS设备成功注册的情况下，能够和Apple TV分享收藏内容。该收藏局限于Clican站点下相关内容。
  • 4, 支持通过iOS设备的本地服务器访问后，能够在Apple TV上浏览iOS设备内的照片
  • 5, 增加龙部落的视频资源
  • 7, 增加优库和人人影视,改内容基本和现有DNS服务器上重复，但可以支持本地代理下载播放，以解决卡顿、缓冲问题
