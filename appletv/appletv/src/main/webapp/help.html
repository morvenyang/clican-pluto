使用帮助


面向的用户群体:

  • 1, 你必须要有iTouch/iPhone/iPad
  • 2, 如果你在使用ATV播放在线视频的时候特别卡，极力推荐你使用该软件的代理下载功能
  • 3, 如果想要播放本地的MKV或MP4文件，但是又不想投入额外的费用购买NAS，推荐你使用该软件的SMB共享
  • 4, 如果你面临迅雷登录问题，可以尝试使用该软件的迅雷登录

安装流程:

  • 1, 通过AppStore搜索《天王星》下载安装
  • 2, 在iOS设备上启动天王星，然后就可以通过Apple TV访问TT DNS服务器(比如180.153.225.136)预告片->Clican->本地服务器，其中本地服务器上的内容就是天王星内置的内容
  • 3, 如果第2步由于未知原因无法成功的话,也可以在TT的个人链接里设置http://<你设备的ip地址>:8080/appletv/local.xml以直接访问应用的内容

开发这个应用的目的和想法:

  • 1, 由于平时特别是周末的时候使用ATV播放优库或人人影视的内容发现特别卡。因此通过该应用实现一个本地服务器以代理下载视频内容。能够很好的解决在线播放的卡顿、缓冲问题。
  • 2, 由于ATV内置的迅雷登录存在种种认为操作或使用不当的问题导致登录失败。因此该应用通过浏览器内嵌的方式来登录迅雷，彻底解决迅雷无法正常登录的问题。
  • 3, 对于不想采购额外NAS服务器的朋友，可以方便的通过该应用共享Windows/Mac/Linux机器上的文件。实现本地转码和播放。
  • 4, 通过本地服务器的方式降低DNS服务器内容流量的压力。
  • 5, 由于ATV3本身无法越狱，因此在其上面实现复杂编程的可能就被封死。唯一的对外交互途径就是HTTP的交互。因此通过把复杂功能在iPhone/iPad上实现然后在通过HTTP交互的方式以ATV3来呈现。

功能列表:

  • 1, 支持通过iOS设备来代理下载视频内容，以解决播放优库和迅雷等相关视频内容卡顿、缓冲的问题
  • 2, 可以通过iOS设备来登录迅雷，以解决迅雷登录问题。目前登录后的迅雷帐号只能在Clican站点下使用
  • 3, 可以通过iOS设备来浏览相关视频内容，收藏视频，在iOS设备成功注册的情况下，能够和Apple TV分享收藏内容。该收藏局限于Clican站点下相关内容。
  • 4, 支持通过iOS设备的本地服务器访问后，能够在Apple TV上浏览iOS设备内的照片
  • 5, 增加龙部落、搜库、56、迅播影院等大量的视频资源,其中龙部落、人人影视和迅播影院以来迅雷离线和迅雷云播
  • 6, 优库和人人影视的内容基本和现有DNS服务器上重复，但可以支持本地代理下载播放，以解决卡顿、缓冲问题
  • 7, 支持通过NAS SMB文件共享的方式来播放MP4、MP3和MKV文件。暂时不支持外挂字幕。通过共享播放MKV时需要用到iOS本地的存储以保存转码后的文件，因此需要注意空间容量是否可以满足MKV大小的需求。转码后文件大约是MKV文件大小的1.1倍。
